# Changelog

Все заметные изменения проекта фиксируются в этом файле.

## [3.1.0] - 2026-05-15

### UI/UX

- Выбор формата перенесён на главный экран в блок ввода ссылки.
- Превью стало чище: в нём остались сведения о видео, папка сохранения, параллельность и авто-открытие.

### Core

- Добавление в очередь использует текущий формат с главного экрана и сразу сохраняет его как значение по умолчанию.
- Статус runtime-инструментов читает версии из manifest без лишних `yt-dlp --version` и `ffmpeg -version` при обычном запуске.
- Служебные процессы на Windows запускаются без всплывающих консольных окон.

### Build/Release

- Версия приложения поднята до `3.1.0`.
- User-Agent обновлён до `VideoDownloaderPro/3.1`.
- Windows onefile exe пересобран.
- Добавлен GitHub Actions workflow для macOS-сборки.
- Совместимость PySide6 расширена диапазоном `PySide6>=6.8.3,<6.12`, чтобы сборка работала на актуальном Python.

## [3.0.0] - 2026-05-11

### UI/UX

- Полный редизайн в стиле premium utility app: тёмный графитовый интерфейс, зелёное главное действие, фиолетовая активная навигация.
- Добавлены разделы: Загрузки, История, Инструменты, Настройки, О приложении.
- Добавлены PreviewCard, DownloadCard, EmptyState, Toast, статусные chips и отдельный QSS `ui/styles/dark.qss`.
- Блок помощи при сетевых ограничениях и `onyshop.tech` сохранён, но стал менее навязчивым.

### Core

- Проект разделён на `app/`, `core/`, `ui/`, `tests/`; `main.py` стал compatibility wrapper.
- Загрузка переведена с `yt_dlp.YoutubeDL` на внешний `yt-dlp` через `QProcess`.
- Добавлен `ToolchainManager` для managed runtime в AppData: `yt-dlp`, `ffmpeg`, `ffprobe`, `manifest.json`.
- Добавлены безопасные staging-обновления: старый runtime не удаляется до успешной проверки нового.
- Добавлены настройки, SQLite-история загрузок, диагностика и отдельные логи `app.log`, `toolchain.log`, `downloads.log`.

### Build/Release

- Windows-сборка переведена на onefile: `dist\VideoDownloaderPro.exe`.
- В exe добавляются fallback `yt-dlp.exe`, `ffmpeg.exe`, `ffprobe.exe`, которые при запуске копируются в AppData runtime.
- Обновлён PowerShell build script с подготовкой fallback-инструментов и проверкой SHA256 для `yt-dlp`.
- Сохранён опциональный `-UseNuitka`.

### Tests

- Добавлены минимальные тесты для путей AppData, парсера прогресса, форматных пресетов, SQLite history и settings.

## [2.5.0] - 2026-02-24

### UI/UX

- Финальная переработка интерфейса: компактная верхняя панель, стабильная сетка настроек, ровные отступы.
- Удален блок истории для более чистого и быстрого рабочего экрана.
- Обновлен скриншот приложения в репозитории.

### Core

- Стабилизирована очередь загрузок и обработка завершения задач.
- Улучшена логика определения ресурсов в собранных билдах (иконка, ffmpeg).
- Сохранены все ключевые улучшения v2: карточки загрузок, асинхронные превью, отмена задач, автопроверка ссылки.

### Build/Release

- Подготовлен Windows x64 one-file executable.
- Подготовлен пакет macOS для распространения.
- Обновлены release-документация и отдельный changelog.
- Добавлен донат-блок в README (USDT TRC20).

## [2.0.0] - 2026-02-24

### Added

- Полноценная очередь загрузок с управлением параллельностью (1-5).
- Новые карточки задач: превью, прогресс, скорость, статус, действия.
- Проверка ссылки до скачивания через отдельный поток.
- Асинхронная загрузка миниатюр без блокировки интерфейса.
- Меню приложения (`Файл`, `Загрузки`, `Инструменты`, `Справка`) и горячие клавиши.
- VPN-инфоблок и подсказки при сетевых ошибках с переходом на `onyshop.tech`.
- Блокировка дублирования учета результатов задачи.
- Авто-открытие скачанного файла по настройке.

### Changed

- Переработан интерфейс: секции, сетка элементов, новые стили, улучшенная читаемость.
- Упрощена структура экрана: удален раздел истории для более чистого UI.
- Улучшен поиск ресурсов в runtime для сборок (`_MEIPASS`, `Resources`, `Frameworks`, рядом с exe).
- Улучшено авто-определение `ffmpeg` (система + bundled бинарники).
- Обновлена логика отмены и завершения задач.
- Сохранение пользовательских настроек в `~/.video_downloader_config.json`.

### Fixed

- Исправлены наложения элементов и проблемы верстки в панели управления.
- Исправлена работа шрифта на macOS (системный шрифт вместо жестко заданного).
- Улучшена обработка закрытия приложения при активных загрузках.
- Исправлены ситуации с неправильным определением финального пути файла после скачивания.

### Build/Release

- Добавлен скрипт сборки macOS: `scripts/build_release.sh`.
- Добавлен скрипт сборки Windows x64: `scripts/build_release_windows.ps1`.
- Добавлен workflow CI для Windows x64: `.github/workflows/build-windows-x64.yml`.
- Поддержана упаковка релизов с иконкой и bundled `ffmpeg` для офлайн-использования.
