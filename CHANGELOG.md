# Changelog

Все заметные изменения проекта фиксируются в этом файле.

## [2.5.0] - 2026-02-24

### UI/UX

- Финальная переработка интерфейса: компактная верхняя панель, стабильная сетка настроек, ровные отступы.
- Удален блок истории для более чистого и быстрого рабочего экрана.
- Обновлен скриншот приложения в репозитории.

### Core

- Стабилизирована очередь загрузок и обработка завершения задач.
- Улучшена логика определения ресурсов в собранных билдах (иконка, ffmpeg).
- Сохранены все ключевые улучшения v2: карточки загрузок, асинхронные превью, отмена задач, автопроверка ссылки.

### Build/Release

- Подготовлен Windows x64 one-file executable.
- Подготовлен пакет macOS для распространения.
- Обновлены release-документация и отдельный changelog.
- Добавлен донат-блок в README (USDT TRC20).

## [2.0.0] - 2026-02-24

### Added

- Полноценная очередь загрузок с управлением параллельностью (1-5).
- Новые карточки задач: превью, прогресс, скорость, статус, действия.
- Проверка ссылки до скачивания через отдельный поток.
- Асинхронная загрузка миниатюр без блокировки интерфейса.
- Меню приложения (`Файл`, `Загрузки`, `Инструменты`, `Справка`) и горячие клавиши.
- VPN-инфоблок и подсказки при сетевых ошибках с переходом на `onyshop.tech`.
- Блокировка дублирования учета результатов задачи.
- Авто-открытие скачанного файла по настройке.

### Changed

- Переработан интерфейс: секции, сетка элементов, новые стили, улучшенная читаемость.
- Упрощена структура экрана: удален раздел истории для более чистого UI.
- Улучшен поиск ресурсов в runtime для сборок (`_MEIPASS`, `Resources`, `Frameworks`, рядом с exe).
- Улучшено авто-определение `ffmpeg` (система + bundled бинарники).
- Обновлена логика отмены и завершения задач.
- Сохранение пользовательских настроек в `~/.video_downloader_config.json`.

### Fixed

- Исправлены наложения элементов и проблемы верстки в панели управления.
- Исправлена работа шрифта на macOS (системный шрифт вместо жестко заданного).
- Улучшена обработка закрытия приложения при активных загрузках.
- Исправлены ситуации с неправильным определением финального пути файла после скачивания.

### Build/Release

- Добавлен скрипт сборки macOS: `scripts/build_release.sh`.
- Добавлен скрипт сборки Windows x64: `scripts/build_release_windows.ps1`.
- Добавлен workflow CI для Windows x64: `.github/workflows/build-windows-x64.yml`.
- Поддержана упаковка релизов с иконкой и bundled `ffmpeg` для офлайн-использования.
